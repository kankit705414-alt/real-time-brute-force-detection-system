# 🛡️ Real-Time Brute Force Detection System

This project simulates brute force login attempts and detects them in real-time using Python.

## Features
- Simulated failed login attempts
- Real-time log monitoring
- Alert system
- Time-based detection
- Basic IP blocking

## Usage

Run detector:
python detect.py --follow --threshold 5

Run simulation:
python simulate_logins.py --failures 8 --delay 1 --ip 192.168.56.10
